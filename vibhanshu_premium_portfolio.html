01
Completed
UDAAN — STUDENT DROPOUT PREDICTION
Python · Scikit-learn · Pandas · SMOTE · Ensemble Models

Problem: Millions of students drop out of school each year — most institutions only notice after it's too late. Early identification changes everything.

  • Built an ML pipeline to predict students at risk of dropping out before it happens
  • Engineered features from attendance, grades, demographics, and behavioral patterns
  • Tackled severe class imbalance (rare dropout events) using SMOTE oversampling
  • Trained ensemble models (Random Forest, Gradient Boosting) for robust predictions
  • Evaluated with ROC-AUC, F1-score, precision & recall — optimized for recall to minimize missed cases
Impact
Enables early-intervention programs, potentially reducing dropout rates for partnered institutions
What Makes It Unique
Imbalance-aware pipeline; optimized threshold tuning for educational context
Tech Tags
Ensemble SMOTE ROC-AUC EdTech
02
Completed
DIABETES RISK ASSESSMENT SYSTEM
Python · Pandas · NumPy · Scikit-learn · Classification Models

Problem: Diabetes affects 537M+ adults globally — most go undiagnosed until complications arise. Predictive screening from routine medical data can save lives.

  • Built a classification model to assess diabetes risk from standard medical attributes
  • Conducted thorough EDA to surface patterns, outliers, and feature distributions
  • Cleaned missing/noisy medical records; applied feature selection for optimal signal
  • Evaluated across multiple classifiers — selected best performing with cross-validation
Impact
Screening tool for preliminary diabetes risk estimation from routine health checkup data
Tech Tags
Healthcare Classification EDA Feature Selection

HAND GESTURE CURSOR CONTROL
Python · OpenCV · Computer Vision · Real-time Processing

Problem: Touchless interfaces are critical for accessibility and hygiene-sensitive environments. Building one with zero external hardware is the challenge.

  • Built a real-time system that maps hand gestures to cursor movement using only a webcam
  • Used OpenCV for live frame capture, hand landmark detection, and gesture classification
  • Implemented gesture-to-action mapping: move, click, scroll without touching hardware
  • Optimized for low-latency real-time performance on consumer-grade hardware
Wow Factor
